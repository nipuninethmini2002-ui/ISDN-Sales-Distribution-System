# ISDN-Sales-Distribution-System
Web-based sales and distribution management system
This project is a web-based sales and distribution management system developed using PHP and MySQL.

## User Roles
- Admin
- HO Admin
- RDC Staff
- Customer
- Delivery Personnel

## Main Features
- User authentication and role-based access control
- Product viewing and order placement
- Order notifications and email integration
- Invoice generation and viewing
- Role-based dashboards

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- PHPMailer

## Setup Instructions
1. Clone or download the repository
2. Import `database.sql` into phpMyAdmin
3. Update database credentials in `db.php`
4. Run the project using XAMPP or WAMP server
